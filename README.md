# KRD Telegram Shopping Bot 🛍️

متجر تسوق احترافي على Telegram يعمل بـ Python و aiogram.

## المميزات ✨

- 🏪 **تصميم احترافي** - واجهة مستخدم جميلة وسهلة الاستخدام
- 🛍️ **متجر كامل** - فئات ومنتجات مع صور وأسعار ووصف
- 🛒 **سلة التسوق** - إضافة وحذف وتعديل الكميات
- 📦 **نظام الطلبات** - تتبع الطلبات وحالاتها
- 👤 **حسابات المستخدمين** - تخزين البيانات الشخصية والعناوين
- 🔧 **لوحة تحكم إدارية** - إدارة المنتجات والفئات والطلبات والمستخدمين
- ➕ **إضافة/تعديل/حذف** - منتجات وفئات بسهولة
- 📊 **عرض الطلبات والمستخدمين** - إحصائيات كاملة

## المتطلبات 📋

- Python 3.8+
- pip

## التثبيت 🚀

### 1. نسخ المستودع
```bash
git clone https://github.com/margarettmj4555-byte/krd-telegram-bot.git
cd krd-telegram-bot
```

### 2. إنشاء بيئة افتراضية (اختياري لكن موصى به)
```bash
python -m venv venv
source venv/bin/activate  # على macOS/Linux
# أو
venv\\Scripts\\activate  # على Windows
```

### 3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 4. إعداد متغيرات البيئة
انسخ `.env.example` إلى `.env`:
```bash
cp .env.example .env
```

ثم عدّل ملف `.env` وأضف:
- **BOT_TOKEN**: رمز البوت من @BotFather على Telegram
- **ADMIN_ID**: معرّف Telegram الخاص بك (اليوزر ID)

```env
BOT_TOKEN=YOUR_BOT_TOKEN_HERE
ADMIN_ID=YOUR_USER_ID_HERE
```

### 5. تشغيل البوت
```bash
python main.py
```

## الحصول على BOT_TOKEN و ADMIN_ID 🔑

### الحصول على BOT_TOKEN:
1. افتح Telegram وابحث عن @BotFather
2. أرسل `/start`
3. أرسل `/newbot`
4. اتبع التعليمات واحصل على التوكن

### الحصول على ADMIN_ID:
1. ابحث عن @userinfobot على Telegram
2. ابدأ البوت وسيعطيك معرّفك الفريد

## الاستخدام 💡

### للمستخدمين:
1. ابدأ المحادثة مع البوت: `/start`
2. اختر **🛍️ متجر** للتسوق
3. اختر الفئة ثم المنتج
4. أضف المنتج للسلة
5. انتقل إلى السلة واطلب الشراء
6. تحقق من طلباتك في **📦 طلباتي**

### للمسؤول (Admin):
1. اضغط على أي أمر في القائمة الرئيسية
2. البوت يتعرف عليك كـ Admin تلقائياً
3. يمكنك:
   - ➕ **إضافة فئات** - فئات جديدة للمنتجات
   - ➕ **إضافة منتجات** - منتجات جديدة مع الأسعار
   - 📋 **إدارة الطلبات** - عرض وتحديث حالات الطلبات
   - 👥 **إدارة المستخدمين** - عرض معلومات المستخدمين

## قاعدة البيانات 🗄️

البوت يستخدم SQLite الذي يُحفظ تلقائياً في ملف `krd_bot.db`:

### الجداول:
- **users**: بيانات المستخدمين (الاسم، الرقم، العنوان)
- **categories**: فئات المنتجات
- **products**: المنتجات (مع الأسعار والوصف)
- **cart**: سلة التسوق
- **orders**: الطلبات
- **order_items**: تفاصيل عناصر الطلبات

## الميزات التفصيلية 🎯

### 🛍️ المتجر:
- عرض الفئات بشكل منظم
- عرض المنتجات مع الصور (اختياري)
- اختيار الكمية قبل الإضافة
- عرض تفاصيل كاملة للمنتج

### 🛒 السلة:
- إضافة/حذف المنتجات
- تعديل الكميات
- حساب الإجمالي تلقائياً
- تأكيد الطلب

### 👤 الحساب:
- حفظ البيانات الشخصية
- تسجيل رقم الهاتف
- تسجيل عنوان التوصيل

### 📦 الطلبات:
- تتبع جميع الطلبات
- عرض تفاصيل كل طلب
- معرفة حالة الطلب (pending/completed/cancelled)

### 🔧 لوحة التحكم:
- إدارة سهلة للمنتجات والفئات
- عرض جميع الطلبات
- تحديث حالات الطلبات
- عرض معلومات المستخدمين

## ملاحظات مهمة ⚠️

1. **تأكد من البيانات الصحيحة:**
   - BOT_TOKEN يجب أن يكون صحيح من @BotFather
   - ADMIN_ID يجب أن يكون معرفك الفريد

2. **البيئة:**
   - يفضل استخدام Python 3.8 أو أحدث
   - استخدم بيئة افتراضية (venv)

3. **الأمان:**
   - لا تشارك ملف `.env` مع أحد
   - احتفظ بـ BOT_TOKEN سري

4. **الترخيص:**
   - جميع الملفات مفتوحة المصدر
   - متاحة للاستخدام الحر

## استكشاف الأخطاء 🐛

### المشكلة: البوت لا يستجيب
**الحل:**
- تحقق من صحة BOT_TOKEN
- تأكد من تثبيت جميع المتطلبات: `pip install -r requirements.txt`
- أعد تشغيل البوت

### المشكلة: لا تستطيع الوصول إلى لوحة التحكم
**الحل:**
- تحقق من ADMIN_ID في ملف `.env`
- تأكد من أنك أدخلت معرفك الصحيح
- استخدم @userinfobot للتحقق من معرفك

### المشكلة: خطأ في قاعدة البيانات
**الحل:**
- احذف ملف `krd_bot.db` لإعادة تهيئة قاعدة البيانات
- أعد تشغيل البوت
